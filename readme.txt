== Description ==

This plugin will provide you a review link to Google Places, Facebook, Tripadvisor and LinkedIn which can be used in emails, posts on facebook or for sharing on any other social plattform.

> If you like the plugin, feel free to rate it (on the right side of this page)! :)

== Installation ==

1. Install the plugin like you always install plugins, either by uploading it via FTP or by using the "Add Plugin" function of WordPress.
2. Activate the plugin at the plugin administration page
3. Open the plugins administration page to set your entries for the different social networks.
4. The plugin will provide you an direct Link to Google Places, Facebook and LinkedIn reviews pages for sharing .
